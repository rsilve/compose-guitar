# Chord grid editor

## For dev


Prepare with
```
npm install
```

For dev 
```
npm run start
```

For prepare test 
```
npm run pretest
```

For unit test
```
npm run test
```

For continue test
```
npm run build:compile:watch
npm run test:watch
```

For build for prod
```
npm run build:prod
```

For preview prod version
```
npm run preview
```


## I18N

For extract text to localize
```
npm run localize:extract
```

update files in `xliff/` and then run
```
npm run localize:build
```
